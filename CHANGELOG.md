## [1.0.0] - 2026-02-13

### Added / 新增
- ✨ Initial release of Avalonia mobile auto-update project
  初始版本的 Avalonia 移动端自动更新项目
- 📱 Cross-platform support for Android, iOS, Desktop, and Browser
  支持 Android、iOS、Desktop 和 Browser 跨平台
- 🔄 Auto-update service with progress tracking
  带进度跟踪的自动更新服务
- 🎨 Modern Fluent UI design
  现代化的 Fluent UI 设计
- 🌍 Bilingual interface (Chinese/English)
  双语界面（中文/英文）
- 📦 MVVM architecture implementation
  MVVM 架构实现
- 📖 Comprehensive documentation
  完整的文档说明

### Technical Details / 技术细节
- Built with Avalonia 11.3.12
  使用 Avalonia 11.3.12 构建
- Uses .NET 10.0 SDK
  使用 .NET 10.0 SDK
- Integrated GeneralUpdate packages for update functionality
  集成 GeneralUpdate 包实现更新功能
