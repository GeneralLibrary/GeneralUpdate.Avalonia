## 贡献指南

欢迎通过 GitHub 协作流程参与贡献：

1. Fork 本仓库到你的账号。  
2. 从 `main` 分支创建功能分支：`feature/{{short-description}}`。  
3. 提交代码并确保通过项目既有的构建与测试流程。  
4. 提交 Pull Request，清晰说明变更背景、内容与影响范围。  
5. 根据 Code Review 反馈迭代，合并后删除分支。  

提交前建议：

- 遵循项目现有代码风格与命名规范。  
- 为新增功能或修复补充必要测试。  
- 更新相关文档，确保示例可运行。
